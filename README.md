# AquilaTrace

AquilaTrace is an AI-powered terrorism financing threat detection platform

## Features

- Log ingestion API
- Anomaly detection
- Risk scoring
- Alert generation
- JWT authentication
- SOC dashboard metrics

## Run

Install dependencies:

pip install -r requirements.txt

Start server:

uvicorn app.main:app --reload

Open API docs:

http://127.0.0.1:8000/docs
